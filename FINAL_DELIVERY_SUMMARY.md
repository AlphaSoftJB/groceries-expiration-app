## üéØ How to Get Started

### 1. Clone the Repository
```bash
git clone https://github.com/AlphaSoftJB/groceries-expiration-app.git
cd groceries-expiration-app
```

### 2. Backend Setup
```bash
cd backend-java

# Configure database in application.properties
# (See PRODUCTION_DEPLOYMENT_GUIDE.md)

# Build and run
mvn clean install
mvn spring-boot:run

# Backend runs on http://localhost:8080
# GraphQL: http://localhost:8080/graphql
```

### 3. Frontend Setup
```bash
cd frontend

# Install dependencies
npm install

# Run iOS
npx react-native run-ios

# Run Android
npx react-native run-android
```

### 4. Docker Setup (Easiest)
```bash
# Start all services
docker-compose up -d

# Check status
docker-compose ps

# View logs
docker-compose logs -f
```
