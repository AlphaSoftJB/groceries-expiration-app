# Groceries Expiration Tracking App - Admin Web Panel

A modern, responsive web-based administration dashboard for managing the Groceries Expiration Tracking App.

## Quick Start

```bash
# Install dependencies
npm install

# Start development server (requires backend running on port 8080)
npm run dev

# Access at http://localhost:5173
```

## Features

- Real-time statistics dashboard
- User and household management
- Item tracking and expiration monitoring
- System health monitoring
- GraphQL integration with Java backend

## Configuration

Create `.env` file:
```
VITE_GRAPHQL_URL=http://localhost:8080/graphql
```

## Technology Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Apollo Client
- Lucide React Icons
